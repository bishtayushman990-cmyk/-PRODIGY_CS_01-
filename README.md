# -PRODIGY_CS_01-
Caesar Cipher Encryption Tool

Overview

The Caesar Cipher Encryption Tool is a Python-based application that demonstrates the fundamentals of cryptography using the Caesar Cipher algorithm. The project allows users to encrypt and decrypt messages by applying a customizable shift key. A graphical user interface (GUI) built with Tkinter provides an interactive and user-friendly experience.

Features

- Encrypt plaintext messages using the Caesar Cipher algorithm
- Decrypt encrypted messages using the correct shift key
- Brute Force Attack simulation to demonstrate basic cryptanalysis
- Support for uppercase and lowercase letters
- Preserves numbers, spaces, and special characters
- User-friendly GUI built with Tkinter
- Error handling for invalid inputs

Technologies Used

How It Works

1. Enter a message.
2. Enter a shift value.
3. Choose Encrypt or Decrypt.
4. View the processed output instantly.
5. Use the Brute Force option to test all possible shift values.
